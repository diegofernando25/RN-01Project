1. Crear aplicación

```
npx create-expo-app@latest
```

2. Arrancar de nuevo

```
npx expo start
```

2. Revento la comunicación?

```
Prueba a hacer cambios en el proyecto, en caso de que
no se carguen los cambios en la terminal, entonces se trata del watchman. Prueba estos comandos:

brew install watchman --> Instala o reinstala
watchman watch-del-all --> Limpia watchers
npx expo start -c    ---> Arranca de nuevo

```
